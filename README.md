📊 Zipper Store Sales Analysis | Power BI Project
📌 Project Overview

This project is an interactive Power BI dashboard created to analyze sales performance of a Zipper Store across multiple product categories.

The dashboard contains multiple pages for different categories such as:

Home & Kitchen
Clothing
Medicine
Grocery
Electronics
Sports
Books & Stationary
Toys
Beauty

The project helps understand sales trends, category performance, customer insights, and business KPIs using interactive visualizations.
